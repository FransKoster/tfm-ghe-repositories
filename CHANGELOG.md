## [1.0.0] - 2025-12-11

### Added

- Initial release of GitHub Enterprise Repositories Terraform Module
- Repository creation and management with comprehensive configuration options
- Branch protection rules with status checks and pull request reviews
- Team and collaborator access management
- Deploy keys configuration
- Webhooks integration
- GitHub Actions secrets and variables (repository and organization level)
- GitHub Pages configuration
- Security and analysis features (advanced security, secret scanning, push protection)
- Support for repository archival instead of deletion
- Template repository support
- Comprehensive examples (basic and advanced)
- Full documentation with best practices and troubleshooting guides

### Features

- Manages all repository settings including merge strategies, features, and topics
- Implements branch protection with customizable rules
- Supports team-based and individual collaborator access control
- Configures CI/CD integration through deploy keys and webhooks
- Securely manages secrets and variables at repository and organization levels
- Enables GitHub security features
- Provides flexible output for integration with other modules
